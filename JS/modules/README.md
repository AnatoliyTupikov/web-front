# Modules

Из-за разрошегося js кода, когда в браузере пишут чуть ли не полноценные программы, нужна структуризация.
Модули - это один из ее способов.
Встроенная обработка модулей JavaScript связана с инструкциями [import](/web-front/FrontEnd/JS/modules/scrt.js) и [export](/module.js).

Main statements:
- Обычно хранится с расширением ".mjs"
